# SAT-problem
A benchmark framework for evaluating SAT solvers using various algorithms. 

Algorithms used : Naïve, PPSZ, WalkSAT (Schöning), DPLL, CDCL (picosat)

- CDCL solver used : picosat https://fmv.jku.at/picosat/
- SAT problem database : SATLIB https://www.cs.ubc.ca/~hoos/SATLIB/benchm.html
                                https://www.cs.ubc.ca/~hoos/SATLIB/Benchmarks/SAT/BMC/description.html
